IO modules - HartreeParticleDSL.IO\_modules package
======================================

IO modules handle input and output code generation for IO.

IO modules are created in user scripts, and provided to the
DSL with ``HartreeParticleDSL.set_io_modules``.

Each IO module has its own set of backends that it can support,
check their documentation for more details.
